### 📘 About This Project

This repository hosts semantically structured vocabularies for European Delegated Regulations. Each term is stored in Markdown with rich metadata and exported as SKOS-based RDF for use in Linked Data platforms.

- ✅ Human-readable definitions  
- 🔗 RDF outputs for interoperability  
- 🌐 GitHub Pages publishing
